-<b>quitgame</b> - quits game <br>
-<b>cheat</b> - you can set the cheat var <br>
-<b>flashing - changes the flashing > var <br>
-<b>></b> - changes the flashing > var <br>
-<b>exit</b> - exits the console <br>
